<b>母集団</b>
データを取得する対象となる要素の集合。
平均値や分散などのパラメーターを知りたいデータ全体のこと。

<b>母数</b>
母集団独自のパラメーター。
母平均：母集団の平均値。$\mu$で表される。
母分散：母集団の分散。$\sigma^{2}$で表される。
母標準偏差：母集団の標準偏差。$\sigma$で表される。

<b>標本</b>
母集団から抽出された一部の要素の集合。
標本平均：標本の平均。$\bar{x}$で表される。

#### ■ 問4　標準化
<b>標準正規分布</b>
正規分布において、平均$\mu=0$, 標準偏差$\sigma=1$、すなわち$N(0, 1^{2})$を標準正規分布と呼ぶ。

<b>標準化</b>
データの値から平均値を引き、標準偏差で割る。
データが$N(\mu, \sigma^{2})$に従うとき、
$$u = \frac{x-\mu}{\sigma}$$
は$N(0, 1^{2})$に従う。

データそのものの値に依存せず、平均値と標準偏差（分散）を考慮した数字の大きさを算出できる。
例：偏差値
$$
偏差値 = \frac{得点 - 平均点}{標準偏差} × 10 ＋ 50
$$

解答
1.正規分布のうち、平均μ=0、標準偏差σ=1のものを標準正規分布と呼ぶ。

2.各データから平均値$\mu$を引いて、標準偏差$\sigma$で割る。

#### ■ 問5　正規分布の性質
<b>正規分布の性質</b>
①データが$N(\mu, \sigma^{2})$に従い、$a, b$を定数とするとき、
$ax + b$は$N(a\mu+b, a^{2}\sigma^{2})$に従う。

証明：https://ai-trend.jp/basic-study/normal-distribution/nature/

②データ$x_{1}$とデータ$x_{2}$が互いに独立にそれぞれ$N_{1}(\mu_{1}, \sigma^{2}_{1})$、$N_{2}(\mu_{2}, \sigma^{2}_{2})$に従うとき、
$x_{1} + x_{2}$は$N(\mu_{1} + \mu_{2}, \sigma^{2}_{1} + \sigma^{2}_{2})$に従う。（正規分布の再生性）

証明：https://ai-trend.jp/basic-study/normal-distribution/nature/

③①と②をより一般的にして、データ$x_{1}, x_{2}, ..., x_{n}$が互いに独立にそれぞれ$N_{i}(\mu_{i}, \sigma^{2}_{i})$に従うとき、
$a_{1}x_{1} + a_{2}x_{2} + ...+a_{n}x_{n}$は$N(a_{1}\mu_{1} + a_{2}\mu_{2} + ... + a_{n}\mu_{n}, a^{2}_{1}\sigma^{2}_{1} + a^{2}_{2}\sigma^{2}_{2} + ... + a^{2}_{n}\sigma^{2}_{n})$に従う。

解答
1.$ax_{1} + b$は$N(a\mu+b, a^{2}\sigma^{2})$に従う。

2.$x_{1} + x_{2}$は$N(\mu_{1} + \mu_{2}, \sigma^{2}_{1} + \sigma^{2}_{2})$に従う。

3.$a_{1}x_{1} + a_{2}x_{2} + ...+a_{n}x_{n}$は$N(a_{1}\mu_{1} + a_{2}\mu_{2} + ... + a_{n}\mu_{n}, a^{2}_{1}\sigma^{2}_{1} + a^{2}_{2}\sigma^{2}_{2} + ... + a^{2}_{n}\sigma^{2}_{n})$に従う。

#### ■ 問6　統計量の分布

得られるデータは平均値や分散を持つ分布に従って現れた値とみなすことができる。
データから算出される統計量もまた、平均値や分散を持つ分布に従う。

<b>標本平均の分布</b>
データが互いに独立に$N(\mu, \sigma^{2})$に従うとき、
$\bar{x}$は$$N(\mu, \frac{\sigma^{2}}{n}) \quad (n:標本数)$$に従う。

証明：http://risalc.info/src/st-normal-distribution-summary.html#sampleMean

<b>標本分散の分布</b>
データが互いに独立に$N(\mu, \sigma^{2})$に従うとき、
$$\frac{S}{\sigma^{2}}は自由度n-1の\chi^{2}分布 \quad (S:平方和)$$
に従う。
※母分散の推定の回で扱います。

証明：http://joe.bayesnet.org/?p=5052

<b>標本平均の標準化</b>
データが互いに独立に$N(\mu, \sigma^{2})$に従うとき、
（$\bar{x}$が$N(\mu, \frac{\sigma^{2}}{n})$に従うとき）
$$u = \frac{\bar{x}-\mu}{\sqrt{\frac{\sigma^{2}}{n}}}$$
は$N(0, 1^{2})$に従う。

解答
1.平均値$\bar{x}$は$N(\mu, \frac{\sigma^{2}}{n})$に従う

2.平均値$\bar{x}$から$\bar{x}$の平均値$\mu$を引き、標準偏差$\sqrt{\frac{\sigma{2}}{n}}$で割る。


#### ■ 問7　統計理論の基本定理

統計理論の基本定理
※確率変数$x$が正規分布に従っていなくても成り立つ

<b>①チェビシェフの不等式</b>
確率変数$x$について、平均値が$\mu$、標準偏差が$\sigma$のとき、
$$Pr\{|x-\mu| \geq k\sigma\} \leq \frac{1}{k^{2}} \quad(Pr\{\}:確率、k:定数、k>0)$$
（xが平均値$\mu$から$k\sigma$以上離れた値をとる確率は$\frac{1}{k^{2}}$以下）

証明：http://www.allisone.co.jp/html/Notes/Mathematics/Probability/Chebyshev's-inequality/index.html

<b>②大数の法則</b>
確率変数$x_{i}(i=1,2,...,n)$が互いに独立で、平均値が$\mu$、標準偏差が$\sigma$のとき、
$n\rightarrow\infty$とすると、
$$\bar{x}\rightarrow\mu$$

（標本の数を無限大にすると、$\bar{x}$の値は真の値$\mu$に近づく）

証明：https://k-san.link/law-of-large-numbers/

<b>③中心極限定理</b>
確率変数$x_{i}(i=1,2,...,n)$が互いに独立で、平均値が$\mu$、標準偏差が$\sigma$であり、$n$が十分に大きいとき、
$$\bar{x}は近似的にN(\mu, \frac{\sigma^{2}}{n})に従う$$

($u = \frac{\bar{x}-\mu}{\sqrt{\sigma^{2}/n}}はN(0, 1^{2})$に従う。)

証明：https://academ-aid.com/statistics/thm-clt
